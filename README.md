# LUXE E-commerce

A full-stack e-commerce application.

## Features
- User Authentication (Login/Register)
- Product browsing and searching
- Shopping Cart functionality
- Admin Dashboard for managing products and orders

## Technologies Used
- Frontend: React, Vite
- Backend: Node.js, Express, MongoDB

## How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rajanikanth45/luxe-ecommerce.git
   cd luxe-ecommerce
   ```

2. **Install Dependencies:**
   - For backend: `cd backend && npm install`
   - For frontend: `cd frontend && npm install`

3. **Set up Environment Variables:**
   Create a `.env` file in the `backend` folder and add your specific configurations (e.g., MongoDB URI, JWT Secret).

4. **Start the Development Servers:**
   - Backend: In the `backend` folder, run `npm run dev`
   - Frontend: In the `frontend` folder, run `npm run dev`
